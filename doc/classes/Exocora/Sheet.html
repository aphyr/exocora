Class Exocora::Sheet
In: lib/exocora/sheet.rb
Parent: Object

A single CGI script. Requests are processed through a chain of actions

sanitize - extract and sanitize data from the CGI request process - do stuff render - render to the specified template.

Public Instance methods

output(string)

Sends output to the client. The first time output is called, it sends an HTTP header first.

process(data)

Process is the meat of the script, which takes the clean data from sanitize. It returns a hash like {:a => 2}, which become instance variables @a = 2 in the template.

render(variables = {})

Renders the erubis template for this action. Takes a hash of variables to render. The default template is determined by underscoring this sheet‘s class name, but another template can be specified using template.

run()

This is the action which initiates processing of the script.

sanitize(cgi)

Sanitize takes a CGI request object as its single parameter. It returns a hash of sanitized data, which is passed to the process method.

template(template)

Sets the name of the template to render.
